**In order to run fixtures:**

1. ```./manage.py reset_db```
2. ```./manage.py migrate``` make sure you ```makemigrations``` first and then migrate the models in.

3. not sure what the linux/mac cmd is but run ```$ bash utility/load_data.sh```
